數據管理分析師Data Management Analyst 11/08更新

統一資訊股份有限公司 本公司其他工作
應徵
11~30人應徵

工作內容

接觸團隊數據系統、數據產品，除了發揮數據洞察思維進行數據分析外，需要深入了解目前團隊運行的數據ETL或者數據產品的運作模式。維護團隊自有數據系統運作，並利用雲端資源、機器學習、深度學習甚至是生成式AI為客戶打造便利的數據產品。 【工作內容】 1. 執行數據導入、清理資料、數據轉換、資料維護等工作。 2. 了解團隊系統的資料流及資料結構，設計彈性化資料表內容。 3. 維護團隊自有數據產品或數據系統。 4. 開發具自動化或系統化性質的數據產品或應用服務。 5. 透過創新及富有彈性的心態進行數據分析，執行管理層指派主題分析任務，並對結果進行報告說明。 6. 自主研究數據，透過視覺化儀表板或簡報，導引出數據可用的商業價值，供團隊決策使用與方案建議。 7. 依不同情境或專業擔任客戶教育訓練講師。 8. 其他主管交辦事項。 【經驗需求】 1. 國、內外大學(含)以上畢業，具商業、統計、數學、資訊等相關科系畢業尤佳。 2. 具備數據分析概念或經驗，能夠有效地使用數據分析工具進行統計分析、趨勢分析等。 3. 有使用ETL工具的經驗，了解數據轉換過程並且能夠設計和實施相關流程。 4. 有建立模型、爬蟲或機器學習、深度學習等實作經驗。 【技能需求】 1. 熟悉Tableau、Power BI、Excel、PowerPoint等工具。 2. 熟悉至少兩項程式語言能力，Ex. SQL、Python、R等。 3. 熟悉機器學習、深度學習等演算法概念。 4. 熟練運用數據分析工具，如市場分析、情報收集、競品分析等，並進行資訊彙整收斂。 5. 能夠參與並實施企業戰略，理解公司目標，提出相應的行動計劃。 * 若有數據分析或相關簡報作品煩請一併附上。 Data Management Analyst In addition to utilizing a data-driven mindset for analysis, it is essential to gain in-depth understanding of the current operational model of the team's data ETL or data product. This involves maintaining the team's proprietary data systems and leveraging cloud resources, machine learning, deep learning, and even generative AI to craft convenient data products for clients. 【Job Description】This position is part of the Data Center Team. 1.Execute data import, clean data, perform data transformation, and maintain data. 2.Understand the data flow and structure of the team's systems, design flexible data table contents. 3.Maintain the team's proprietary data products or data systems. 4.Develop automated or systematic data products or application services. 5.Conduct data analysis with an innovative and flexible mindset, execute theme analysis tasks assigned by management, and report and explain the results. 6.Independently research data and present insights through visual dashboards or presentations, guiding the team's decision-making. 7.Serve as a client education and training instructor in different contexts or specialties. 8.Handle other tasks assigned by supervisors. 【Experience】 1.Bachelor's degree or above in business, statistics, mathematics, information technology, or related fields. 2.Understanding of data analysis concepts or experience, effective use of data analysis tools for statistical analysis, trend analysis, etc. 3.Experience using ETL tools, understanding the data transformation process, and designing and implementing related processes. 4.Practical experience in model building, web scraping, machine learning, deep learning, etc. 【Skills】 1.Proficiency in tools like Tableau, Power BI, Excel, PowerPoint. 2.Proficiency in at least two programming languages, e.g., SQL, Python, R. 3.Familiarity with concepts of machine learning, deep learning, and algorithms. 4.Proficient in using data analysis tools for market analysis, intelligence gathering, competitive analysis, etc. 5.Ability to participate in and implement enterprise strategies, understanding company goals, and proposing corresponding action plans. ＊If you have any data analysis or related presentation portfolios, please include them as well.
